## Smart-Grid

There are some key features in this project which focuses on a self managing engine related to the national infrastructure during the times of extreme energy shortages. Now features like a hierarchical mapping system which organizes the whole physical grid structure like a family tree which connects big power stations directly down to the individual homes. Further this whole engine acts like an self operating judge, where the decision like whether to give or cut the power is properly calculated by combining the data of real time bill payment with the digital theft detection part. So as a result this project replaces the static and the manual load shedding schedules, with a smart optimization engine which balances the limited supply of electricity by prioritizing the areas based on their verified revenue recovery and the absence of any infrastructure interference.
